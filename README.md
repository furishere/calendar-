## Overview

This project is a responsive calendar UI built using React and Tailwind CSS. It displays a monthly calendar layout along with a notes section, focusing on clean design and responsive behavior across different screen sizes.

---

## Features

* Responsive layout (mobile-first design)
* Calendar grid with days and dates
* Notes section for additional content
* Reusable React components
* Clean UI using Tailwind CSS

---

## Tech Stack

* React (Vite)
* Tailwind CSS
* JavaScript (ES6+)

---

## Project Structure

* `Calendar.jsx` – Main calendar layout
* `Notes.jsx` – Notes section component
* `HeadName.jsx` – Reusable heading component
* `assets/` – Images used in the UI

---

## Design Choices

* Used Tailwind CSS for fast and consistent styling
* Followed a mobile-first approach to ensure usability on phones
* Component-based structure for reusability and maintainability
* Flexbox and Grid used for layout management
* Avoided fixed heights to improve responsiveness

## How to Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/furishere/calendar-.git
```

### 2. Navigate to the project folder

```bash
cd calendar-
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

### 5. Open in browser

Go to:

```
http://localhost:5173
```
